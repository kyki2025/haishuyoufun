# 海叔有FUN个人网站

🌊 50岁大叔的数字生活实验室 - 有趣 · 有范 · 有料

## 🚀 项目介绍

这是海叔的个人网站，展示原创AI工具、生活分享和技术心得。网站采用现代化的React + TypeScript技术栈构建，设计简洁美观，用户体验友好。

## ✨ 主要功能

### 🛠️ 原创工具
- **海叔PDF转换器**: 专业的PDF文件格式转换工具
- **海叔品茶记录**: 茶文化爱好者的品茶记录管理工具

### 📝 内容展示
- 个人介绍和技术背景
- 生活分享和兴趣爱好
- 工具使用教程和心得

### 🎨 设计特色
- 响应式设计，支持各种设备
- 渐变色彩和现代化UI组件
- 流畅的页面切换动画
- 优雅的用户交互体验

## 🛠️ 技术栈

- **Frontend**: React 18 + TypeScript
- **Styling**: Tailwind CSS + shadcn/ui
- **Icons**: Lucide React
- **Routing**: React Router
- **Build Tool**: esbuild

## 📦 安装和运行

1. 克隆仓库
```bash
git clone https://github.com/your-username/haishu-fun-website.git
cd haishu-fun-website
```

2. 安装依赖
```bash
npm install
```

3. 启动开发服务器
```bash
npm run dev
```

4. 构建生产版本
```bash
npm run build
```
